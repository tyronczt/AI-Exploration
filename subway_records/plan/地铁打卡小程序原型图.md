
# 地铁打卡小程序原型图

**文档说明**：本文档基于 `地铁打卡小程序原型图_设计说明.md` 创建，旨在通过文字描述和ASCII艺术图的形式，清晰展示小程序各页面的布局、组件及交互元素，为开发团队提供准确的可视化参考。文档中的页面结构、组件命名及交互逻辑均与设计说明保持一致。

**当前日期**：2026年02月07日

---

## 页面总览

小程序主要包含以下核心页面：

1.  **首页 (Home Page)**：用户登录后的主界面，展示个人数据、快速打卡入口及站点列表。
2.  **打卡详情页 (Check-in Detail Page)**：展示单次打卡的详细信息、状态及成就。
3.  **站点探索页 (Station Explore Page)**：以地图或列表形式展示所有可打卡站点及信息。
4.  **个人中心页 (Profile Page)**：管理用户账户、设置、成就勋章及历史记录。
5.  **打卡成功页 (Check-in Success Page)**：打卡成功后的即时反馈页面。

---

## 1. 首页 (Home Page)

**页面功能**：数据概览与核心操作入口。

### 布局结构与ASCII艺术图

```
+---------------------------------------------------+
|  🚇 地铁打卡                 [消息图标] [设置图标] |
+---------------------------------------------------+
|                                                   |
|         [用户头像]                                |
|          用户名                                   |
|          “已连续打卡 15 天”                       |
|                                                   |
+---------------------------------------------------+
|                                                   |
|      +----------------------+                     |
|      |    📍 当前所在位置    |                     |
|      |   [XX地铁站]         |                     |
|      |                      |                     |
|      |   [立即打卡] 按钮     |                     |
|      +----------------------+                     |
|                                                   |
+---------------------------------------------------+
|  📊 本周数据统计                                  |
|  +---------+  +---------+  +---------+            |
|  |  打卡数  |  |  里程数  |  |  成就数  |            |
|  |    12   |  |   85km  |  |    8    |            |
|  +---------+  +---------+  +---------+            |
|                                                   |
+---------------------------------------------------+
|  🗺️ 最近访问站点                                  |
|  [站点A卡片] [站点B卡片] [站点C卡片] ...         |
|  (横向滚动列表)                                   |
|                                                   |
+---------------------------------------------------+
|  [底部导航栏]                                     |
|  首页 | 探索 | 打卡 | 我的                        |
+---------------------------------------------------+
```

### 组件描述
1.  **顶部导航栏**：
    *   **左**：应用Logo与名称“🚇 地铁打卡”。
    *   **右**：消息铃铛图标、设置齿轮图标。
2.  **用户信息区**：
    *   居中显示用户头像、用户名。
    *   下方显示激励文案，如“已连续打卡 X 天”。
3.  **核心打卡区**：
    *   显示“📍 当前所在位置”标题。
    *   动态显示检测到的或用户选择的 `[XX地铁站]` 名称。
    *   突出的 `[立即打卡]` 主按钮。
4.  **数据统计区**：
    *   标题“📊 本周数据统计”。
    *   并排三个数据卡片：打卡数、里程(km)、成就数，均包含图标和数字。
5.  **最近访问站点区**：
    *   标题“🗺️ 最近访问站点”。
    *   横向滚动列表，包含多个站点信息卡片（显示站点名、图片缩略图）。
6.  **底部导航栏**：
    *   固定位置：`首页`(高亮)、`探索`、`打卡`(可能为相机图标)、`我的`。

### 交互说明
*   点击`[立即打卡]`按钮，跳转至**打卡详情页**。
*   点击顶部`[设置图标]`，跳转至**个人中心页**的设置部分。
*   点击`[最近访问站点]`的任一卡片，跳转至该站点的**打卡详情页**。
*   点击底部导航栏切换对应主页面。

## 2. 打卡详情页 (Check-in Detail Page)

**页面功能**：展示特定打卡记录的详细信息。

### 布局结构与ASCII艺术图

```
+---------------------------------------------------+
|  < 返回                                  ⋮ (更多) |
+---------------------------------------------------+
|                                                   |
|            [站点大型封面图]                       |
|                                                   |
|            【站点名称】                           |
|            ⭐⭐⭐⭐⭐ (4.8)                     |
|                                                   |
+---------------------------------------------------+
|  📝 打卡记录                                      |
|  时间：2026-02-07 18:30                          |
|  状态：✅ 打卡成功                                |
|  备注：[用户输入的备注文本...]                   |
|  [编辑] 按钮                                      |
|                                                   |
+---------------------------------------------------+
|  🏆 本次解锁成就                                  |
|  +-------------------+                            |
|  | [成就图标]        |                            |
|  | “首访者”          |                            |
|  +-------------------+                            |
|                                                   |
+---------------------------------------------------+
|  🗺️ 站点信息                                      |
|  线路：1号线                                      |
|  特色：“经典站台，百年风貌”                       |
|  [查看详情] 按钮                                  |
|                                                   |
+---------------------------------------------------+
|  [分享] 按钮        [收藏] 按钮                   |
+---------------------------------------------------+
```

### 组件描述
1.  **顶部导航栏**：
    *   返回箭头、页面标题（通常为站点名，此处简化为导航栏）、更多操作“⋮”图标。
2.  **站点视觉区**：
    *   占据显著位置的站点封面图。
    *   下方居中显示站点名称和评分。
3.  **打卡记录区**：
    *   标题“📝 打卡记录”。
    *   列表形式展示：打卡时间、状态（成功/失败）、用户备注。
    *   `[编辑]`按钮用于修改备注。
4.  **成就展示区**：
    *   标题“🏆 本次解锁成就”。
    *   展示由此打卡解锁的成就徽章及其名称。
5.  **站点信息区**：
    *   标题“🗺️ 站点信息”。
    *   列出站点所属线路、简短特色描述。
    *   `[查看详情]`按钮可跳转至更丰富的站点百科页面。
6.  **底部操作栏**：
    *   `[分享]`与`[收藏]`按钮并排显示。

### 交互说明
*   点击顶部`< 返回`，返回上一页（通常是首页）。
*   点击`[编辑]`按钮，可对打卡备注进行内联编辑。
*   点击`[查看详情]`按钮，跳转至站点百科页（非本原型范围）。
*   点击`[分享]`或`[收藏]`，触发相应功能。

## 3. 站点探索页 (Station Explore Page)

**页面功能**：发现和浏览所有可打卡站点。

### 布局结构与ASCII艺术图

```
+---------------------------------------------------+
|  站点探索                    [搜索图标] [筛选图标]|
+---------------------------------------------------+
|                                                   |
|  [地图视图] 图标      [列表视图] 图标             |
|  (当前选中列表视图)                               |
|                                                   |
+---------------------------------------------------+
|  +--------------------------------------------+   |
|  |  🔍 搜索站点或线路...                      |   |
|  +--------------------------------------------+   |
|                                                   |
+---------------------------------------------------+
|  📍 1号线                                        |
|  +-------------------------+  +----------------+  |
|  | [站A图]                 |  | [站B图]        |  |
|  | 站A名称                 |  | 站B名称        |  |
|  | ⭐4.5                   |  | ⭐4.8          |  |
|  | [已打卡] 标签           |  |                |  |
|  +-------------------------+  +----------------+  |
|                                                   |
|  📍 2号线                                        |
|  +-------------------------+                     |
|  | [站C图]                 |                     |
|  | 站C名称                 |                     |
|  | ⭐4.9                   |                     |
|  +-------------------------+                     |
|                                                   |
+---------------------------------------------------+
|  [底部导航栏]                                     |
|  首页 | 探索 | 打卡 | 我的                        |
+---------------------------------------------------+
```

### 组件描述
1.  **顶部导航栏**：
    *   页面标题“站点探索”。
    *   右侧为`[搜索图标]`和`[筛选图标]`。
2.  **视图切换器**：
    *   `[地图视图]`和`[列表视图]`图标，高亮显示当前视图。
3.  **搜索框**：
    *   占位符为“🔍 搜索站点或线路...”。
4.  **站点列表区**：
    *   按地铁线路分组，如“📍 1号线”。
    *   每行显示2个站点卡片（网格布局）。
    *   每个卡片包含：站点图片、名称、评分。
    *   若用户已打卡该站点，卡片上会显示`[已打卡]`标签。

### 交互说明
*   点击`[搜索图标]`或搜索框，聚焦并进入搜索页面。
*   点击`[筛选图标]`，弹出筛选条件面板（如按线路、打卡状态筛选）。
*   点击`[地图视图]`图标，页面切换为地图模式（需集成地图SDK）。
*   点击任意`站点卡片`，跳转至该站点的**打卡详情页**。

### 组件描述
1.  **顶部导航栏**：
    *   页面标题“个人中心”。
    *   右侧`[设置]`图标。
2.  **个人信息头图**：
    *   用户头像、用户名、等级标签（如“地铁旅行家 Lv.7”）。
    *   `[编辑资料]`按钮。
3.  **核心数据概览**：
    *   并排两个数据卡片：`总打卡数`、`总里程`。
4.  **成就展示区**：
    *   标题“🏆 我的成就”。
    *   横向滚动排列已获得的成就徽章。
5.  **历史记录区**：
    *   标题“📁 我的足迹”。
    *   按月分组的打卡记录卡片，显示月份和当月打卡站点总数，右侧有箭头“>”。
6.  **操作区**：
    *   底部明显的`[退出登录]`按钮。
7.  **底部导航栏**：
    *   `我的`标签高亮。

### 交互说明
*   点击`[设置]`图标，进入应用设置子页面。
*   点击`[编辑资料]`按钮，进入资料编辑页面。
*   点击任一`成就徽章`，查看该成就的详细说明。
*   点击月度`足迹卡片`，查看该月的详细打卡列表。
*   点击`[退出登录]`，弹出确认框，确认后清除登录状态并返回登录页。

### 布局结构与ASCII艺术图

```
|            打卡成功！                             |
|                                                   |
|            “你已成功点亮【XX地铁站】”             |

+---------------------------------------------------+
|                                                   |
|         +----------------------+                  |
|         | [本次打卡成就徽章]   |                  |
|         |   “探索先锋”         |                  |
|         +----------------------+                  |
|                                                   |
+---------------------------------------------------+
|                                                   |
|      +------------------------+                   |
|      |    📍 下一个推荐站点    |                   |
|      |   [YY地铁站]           |                   |
|      |   “距离此地 2.3km”     |                   |
|      |                        |                   |

|      +------------------------+                   |
|                                                   |

|      [查看详情] 按钮     [分享喜悦] 按钮          |
|                                                   |
+---------------------------------------------------+
```

### 组件描述
1.  **庆祝氛围区**：
    *   使用大型庆祝emoji（🎉✨）和“打卡成功！”大字标题。
    *   下方有一句个性化文案，包含刚打卡的站点名。
2.  **成就展示区**：
    *   居中展示本次打卡解锁的成就徽章及其名称。
3.  **推荐引导区**：
    *   标题“📍 下一个推荐站点”。
    *   显示推荐站点`[YY地铁站]`的名称和距离信息。
    *   一个次要按钮`[前往探索]`。
4.  **主操作区**：
    *   并排两个主要按钮：`[查看详情]`和`[分享喜悦]`。

### 交互说明
*   页面通常有自动出现的庆祝动画效果。
*   点击`[查看详情]`按钮，跳转至本次打卡的**打卡详情页**。
*   点击`[分享喜悦]`按钮，弹出分享面板，将成功结果分享至微信好友或朋友圈。
*   点击`[前往探索]`按钮，跳转至推荐站点的**站点探索页**或详情页。
*   点击页面背景或等待3-5秒后，页面可能自动关闭并返回**首页**。

## 全局组件与设计规范

1.  **颜色主题**：
    *   **主色**：地铁标志蓝色 (`#007AFF`) 用于主按钮、高亮文字。
    *   **辅助色**：绿色 (`#34C759`) 表示成功状态，橙色 (`#FF9500`) 用于警告或次要操作。
    *   **背景色**：浅灰色 (`#F5F5F7`) 作为页面背景，白色 (`#FFFFFF`) 作为卡片背景。
2.  **字体**：
    *   默认使用系统字体。标题使用加粗或较大字号。
3.  **图标**：采用简洁的线性图标风格，与微信小程序原生图标风格保持一致。
4.  **交互状态**：所有按钮需有`正常`、`按下`、`禁用`三种状态的颜色或透明度变化。
5.  **加载与反馈**：网络请求时显示加载动画，操作成功/失败有相应的Toast提示。
