# GitHub 一周热点

自动获取 GitHub 热门开源项目，生成包含中文摘要、星标趋势图的精美 HTML 日报/周报，并提供日历导航和趋势聚合看板。

## 安装

```bash
npx skills add tyronczt/AI-Exploration/github_daily
```

安装后 skill 目录包含以下文件：

| 文件 | 用途 |
|------|------|
| `SKILL.md` | 技能定义文件，包含完整的工作流程和 AI 指令 |
| `template.html` | 暗色仪表盘风格报告模板 |
| `index.html` | 日历导航首页 + 趋势聚合看板 |
| `build.js` | 一键构建脚本：扫描报告并将数据内嵌到 index.html |

## 环境要求

- **Node.js 14+**：`build.js` 脚本需要 Node.js 运行环境
- 无需安装额外的 npm 依赖，脚本仅使用 Node.js 内置模块

## 使用方式

在 AI 编程助手中调用 `github-trending` 技能，或输入类似指令：

- "获取本周 GitHub 热门项目"
- "生成 GitHub 日报"
- "看看最近 GitHub 上什么项目最火"
- "生成 0606、0607、0608 的日报"

技能会自动完成以下全部流程，**无需手动执行任何命令**：

1. 抓取 GitHub Trending 数据
2. 为每个项目生成中文深度介绍
3. 输出可视化 HTML 日报
4. 自动更新导航首页 `index.html`（内嵌数据，可直接打开）

## 报告内容

每个项目包含：

- 中文短标题和一句话概括（是什么 + 核心特点 + 适合谁）
- 2-3 段博客风格的中文详细介绍
- 最近 7 天星标增长趋势图（基于 Chart.js，含 CSS 降级方案）
- 项目基础数据：Star 数、Fork 数、编程语言
- 可点击的项目链接直接跳转 GitHub

提供暗色仪表盘主题模板，包含深色背景、金银牌排名、趋势条形图和语言分布统计。

## 导航首页与趋势看板

安装后，`index.html` 提供两个视图：

- **日历浏览**：按月展示报告，有报告的日期绿色高亮，点击直接跳转
- **趋势看板**：上榜频次排行、累计星标增长排行、语言热度分布

### 自动构建

每次生成报告时，AI 智能体会**自动运行 `build.js`** 更新导航首页，无需手动操作。生成的 `index.html` 是自包含文件，可以直接双击打开，也可以上传到任意 Web 服务器。

> 如果需要手动重建（如批量导入历史报告后），也可以在 skill 目录运行 `node build.js`。

## 部署到服务器

如果你有自己的服务器或静态网站托管（如 GitHub Pages、Vercel、Cloudflare Pages）：

1. 将 skill 目录中的所有文件上传到服务器的目标路径
2. 在服务器上运行一次 `node build.js` 构建首页（后续由 AI 智能体自动更新）
3. 访问对应 URL 即可看到导航首页

示例（假设部署到 `/var/www/github_daily/`）：

```bash
# 上传文件后
cd /var/www/github_daily/
node build.js
# 然后配置 Web 服务器指向该目录即可
```

## 定时自动更新

可配合 AI 编程助手的定时任务功能，设置每天自动运行：

- 每天定时生成日报
- 自动运行 `build.js` 更新导航首页
