# 词小伴项目结构

## 项目概述
本项目是一个英语单词学习小程序“词小伴”，包含后端API服务和前端微信小程序两个部分。

## 目录结构

## 技术栈

### 后端
- SpringBoot 3.0
- MyBatis Plus
- MySQL 8.0
- Spring Security
- Swagger/OpenAPI 3.0
- Java 17

### 前端
- 微信小程序原生开发

## 功能模块

### 后端API
- 用户管理（注册、登录、个人信息）
- 词库管理（词库分类、单词管理）
- 学习功能（单词学习、测试、复习）
- 统计分析（学习进度、成绩统计）

### 前端小程序 - 词小伴
- 首页（学习统计、推荐词库）
- 词库页（词库列表、词库详情）
- 学习页（单词学习、测试）
- 个人中心（个人信息、学习记录、收藏夹）

## 运行说明

### 后端运行
1. 确保MySQL数据库已安装并运行
2. 创建数据库 `english_learning`（可参考技术方案文档中的SQL脚本）
3. 修改 `back/src/main/resources/application.yml` 中的数据库连接信息
4. 在back目录下执行 `mvn spring-boot:run` 启动后端服务
5. 访问 `http://localhost:8080/api/swagger-ui.html` 查看API文档

### 前端运行
1. 确保已安装微信开发者工具
2. 使用微信开发者工具打开front目录
3. 配置小程序AppID
4. 运行小程序
